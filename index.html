🚨 De-Indexation Analysis REQUIRES IMMEDIATE ACTION
The site has been removed from Google's search index — this is not a rankings drop, it's a complete disappearance.

A normal rankings drop means lower positions but still visible. De-indexation means Google has stopped showing the site entirely. At zero impressions, no amount of search volume helps — the site simply doesn't appear for any query. This has been ongoing since approximately February 2026.

Likely Causes — What to Investigate

🔍 Technical Checks (Do These First — Today)

  • Check Google Search Console for Manual Actions (Penalties tab) — if there's a manual action, it will show here
  • Run site:yourbellplumbing.com in Google — if fewer than 10 results appear, the site is de-indexed
  • Check robots.txt at yourbellplumbing.com/robots.txt — did someone add "Disallow: /" blocking all crawling?
  • Check for noindex meta tags sitewide — a WordPress plugin or site migration can accidentally add noindex to all pages
  • Check GSC Coverage report — look for "Excluded" pages with reasons like "noindex" or "crawl anomaly"
  • Check if the site was migrated, rebuilt, or had major changes around December 2025–January 2026

⚠️ Algorithmic Causes (If No Technical Issue Found)

  • Google ran a Helpful Content Update in late 2024 — thin/AI-generated content pages can be demoted or removed
  • Spam links or negative SEO — check Google Search Console's Links report for unusual link spikes
  • Duplicate content issues — if pages are similar to other sites or duplicated internally
  • Core algorithm update: Google released core updates in Jan and Mar 2026 — site may have been caught
  • GBP/local spam signals — spammy local SEO tactics can trigger manual reviews
  • The GSC property only covers https://yourbellplumbing.com/ — confirm www vs. non-www is consistent
⚠️ Important: The gradual collapse (not overnight) suggests an algorithmic issue or technical issue discovered gradually as Google re-crawled pages — NOT an accidental overnight robots.txt block (which would be instant).

However, a noindex tag added around Jan 1–5 could explain the gradual pattern: Googlebot crawls different pages at different intervals, so a noindex on all pages would show up gradually as each page gets re-crawled. This is the most likely scenario and the first thing to check.

CallRail Analysis

⚠️ CallRail: Zero Calls Tracked for This Client

CallRail is connected to the system under the company name "Yourbellplumbing.com" but shows 0 total calls, 0 answered, 0 missed for all time periods checked (Dec 2025 through Jun 2026, and historically).


This means one of three things:

  • CallRail tracking numbers are not placed on the website (the phone number on the site is a direct number, not a CallRail tracked number)
  • The CallRail account exists but no tracking numbers/campaigns have been configured
  • The tracking number was removed from the site at some point and calls are no longer being recorded

Immediate action needed: Verify that the phone number displayed on yourbellplumbing.com (and in Google Ads / GBP) is a CallRail tracking number, not the direct business line. If it's a direct number, all call data is invisible and the client has no insight into how many calls their website and ads generate.

How to Fix CallRail Tracking

  • Log into CallRail → confirm tracking numbers are assigned to yourbellplumbing.com
  • Set up a "Website Visitor" tracking number — dynamically replaces the phone number on-site
  • Set up a separate tracking number for Google Ads (for call-from-ad tracking)
  • Set up a GBP-specific tracking number to measure map pack calls
  • Verify the tracking script is installed in the site header
  • Test by visiting the site and confirming the number shown matches a CallRail number

Why This Matters Right Now

  • Client is spending significantly on paid search — zero visibility into how many calls those ads generate
  • Cannot calculate cost-per-lead without call tracking data
  • Cannot prove ROI of any marketing investment to the client
  • Cannot identify which keywords or campaigns drive the most calls
  • Cannot track seasonal call volume patterns to answer questions like "why did June calls drop?"
  • GA4 /thank-you/ page had 121 sessions — that's form leads, but phone calls are invisible
Priority Action Plan Start Today
The de-indexation is the #1 priority — everything else is secondary until this is resolved.

Every day the site remains de-indexed, the client is paying 100% for traffic that should be coming organically. Additionally, organic rankings that were built over years are decaying further the longer this persists.
